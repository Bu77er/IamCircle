# IamCircle
For the main project piece, I will be creating a game with a shader applied to make it look hatch shaded. This will allow me to show off gameplay programming abilities as well as some graphics programming within the creation of a game. I will be using Unity and C# for the creation of the game as this is the game engine that I have the most experience with. The bulk of the gameplay will include attacks that you can perform against enemies such as a shockwave AOE attack and a single enemy attack. There will be AI enemies that will try to attack you as well. The main limitations come from the creation of the shader and possibly having to make my own tonal art map generator. This will take the bulk of the project time. At the end, there should be a playable demo of the game as well as some video clips to put in a demo reel.
